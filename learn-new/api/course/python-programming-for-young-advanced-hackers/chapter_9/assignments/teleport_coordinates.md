# Mission: Teleport Coordinates (Tuples)

Store teleport positions as tuples and print them.

## What your program should do

1. Create two tuples: `start_position` and `target_position`, each with two numbers (x and y).
2. Use indexing to get the x and y values from each tuple.
3. Print sentences like:
   - `Start position: (0, 0)`
   - `Target position: (5, 10)`

## Hints

- A tuple looks like this: `start_position = (0, 0)`.
- Use `start_position[0]` for x and `start_position[1]` for y.
- Do the same for `target_position`.
