# Useful and Fun VS Code Extensions

## Visual & UX

- [Dracula Official](https://marketplace.visualstudio.com/items?itemName=dracula-theme.theme-dracula) – Popular, eye-friendly dark theme with good contrast for long coding sessions.
- [vscode-icons](https://marketplace.visualstudio.com/items?itemName=vscode-icons-team.vscode-icons) – Adds distinctive icons for files and folders, making large projects easier to scan.
- [Peacock](https://marketplace.visualstudio.com/items?itemName=johnpapa.vscode-peacock) – Tint the whole VS Code window per project so you instantly recognize which workspace you’re in.
- [Indent Rainbow](https://marketplace.visualstudio.com/items?itemName=oderwat.indent-rainbow) – Colors indentation levels so you can spot nesting errors quickly.

## Productivity & Code Quality

- [Prettier – Code formatter](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) – Opinionated formatter that auto-formats on save for consistent style across teams.
- [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) – Lints JavaScript/TypeScript, catching common bugs and style issues as you type.
- [GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens) – Supercharges the built-in Git view with blame annotations, history, and line-by-line authorship.
- [Path Intellisense](https://marketplace.visualstudio.com/items?itemName=christian-kohler.path-intellisense) – Auto-completes file paths in imports and requires.
- [Todo Tree](https://marketplace.visualstudio.com/items?itemName=Gruntfuggly.todo-tree) – Scans your code for TODO/FIXME comments and shows them in a tree view for quick navigation.
- [Better Comments](https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments) – Styles comments (alerts, questions, TODOs) with different colors to make them stand out.

## AI & Smart Suggestions

- [GitHub Copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot) – AI pair programmer that suggests whole lines or functions based on context.
- [IntelliCode](https://marketplace.visualstudio.com/items?itemName=VisualStudioExptTeam.vscodeintellicode) – Microsoft’s AI-assisted completion that improves IntelliSense based on common coding patterns.

## Web Dev & JS/TS

- [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) – Launches a local dev server with live reload for static or simple web projects.
- [JavaScript (ES6) Code Snippets](https://marketplace.visualstudio.com/items?itemName=xabikos.JavaScriptSnippets) – Handy snippets for common JS/TS patterns, speeding up boilerplate.
- [Quokka.js](https://marketplace.visualstudio.com/items?itemName=WallabyJs.quokka-vscode) – “Scratchpad” for JS/TS that evaluates code live and shows results inline as you type.

## Collaboration, APIs, and DevOps

- [Live Share](https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare) – Real-time collaborative editing and debugging directly inside VS Code.
- [REST Client](https://marketplace.visualstudio.com/items?itemName=humao.rest-client) – Send HTTP requests from `.http` files in VS Code, great for testing APIs without Postman.
- [Docker](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-docker) – Manage containers, images, and debug containerized apps from the editor.

## Regex, Markdown, and Writing

- [Regex Previewer](https://marketplace.visualstudio.com/items?itemName=chrmarti.regex) – Side-by-side view that shows live matches while you edit a regular expression.
- [Markdown All in One](https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one) – Shortcuts, TOC, and auto-formatting of Markdown, all in one extension.
- [:emojisense:](https://marketplace.visualstudio.com/items?itemName=bierner.emojisense) – Emoji autocompletion for Markdown, triggered by typing `:` plus a keyword.

## “Just for Fun” (but still useful)

- [Error Lens](https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens) – Makes errors and warnings loud and inline, turning debugging into a bit of a mini-game.
- [Polacode](https://marketplace.visualstudio.com/items?itemName=pnp.polacode) – Take pretty code screenshots for sharing on social media or docs.
- [Power Mode](https://marketplace.visualstudio.com/items?itemName=hoovercj.vscode-power-mode) – Adds particle effects and screen shake while typing for arcade vibes.
