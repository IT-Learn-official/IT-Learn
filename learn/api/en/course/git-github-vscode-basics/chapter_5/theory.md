# GitHub Remotes: Clone, Push, Pull

Your repo can live in two places:

- local (your computer)
- remote (GitHub)

Clone = download a remote repo:

```bash
git clone <repo-url>
```

Push = upload your commits:

```bash
git push
```

Pull = download new commits:

```bash
git pull
```

---

After cloning, Git usually calls the remote `origin`.
You can check it with:

```bash
git remote -v
```
