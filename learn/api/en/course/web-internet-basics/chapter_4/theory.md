# Chapter 4: Servers (The Machines That Answer You)

Servers are machines or software that answer requests from browsers and apps.

By the end of this chapter, you’ll be able to:

- Define what a server is.
- Explain what a web server does.
- Describe how backend apps and databases fit together.

---

## What Is a Server?

A server is a machine or service that listens for requests and sends responses. "Server" can mean the physical/virtual machine or the software running on it (e.g., Nginx, Apache, Node).

> **Key takeaway:** A server listens and answers — it provides the data or service clients request.

---

## Web Server vs Backend App

- **Web server:** serves static files, handles TLS, and often forwards requests to the backend.
- **Backend app:** runs application logic (login, database access, APIs) and builds dynamic responses.

Modern sites combine browser (client), web server (front door), backend app (logic), and database (storage).
