## ✅ Self-Check Questions

After running each query, ask yourself:

1. Did I get the expected columns?
2. Did I remember the semicolon?
3. Did I spell the column names correctly?
4. Did I use proper SQL syntax?

## 🎓 Bonus Challenges

1. **Column Order**: Try selecting the same columns but in a different order. What happens?
2. **All from Hackers**: Write a query to see all data from the `hackers` table.
3. **Mix and Match**: Create a query selecting any 4 columns from the `skills` table.

## 💡 Common Mistakes to Avoid

- ❌ Forgetting the semicolon at the end
- ❌ Misspelling table or column names
- ❌ Using quotes around column names (usually not needed)
- ❌ Forgetting commas between column names

## 🏆 Success Criteria

You've mastered this assignment when you can:

- Write a SELECT \* query without looking at examples
- Select multiple specific columns with proper syntax
- Retrieve data from different tables confidently
- Identify and fix simple syntax errors
