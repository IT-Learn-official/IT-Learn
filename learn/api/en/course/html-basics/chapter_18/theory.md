# SEO Basics

SEO stands for Search Engine Optimization. It basically means making your website easier to understand for search engines like Google, Bing, etc. This helps your website to rank higher in search results. This means more people can find your website.

---

How to do SEO with HTML:
- Use semantic HTML elements like `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>` to structure your content.
- Include a descriptive meta description in the `<head>` section of your HTML.
- Use heading tags (`<h1>` to `<h6>`) properly to indicate content hierarchy.
- Add alt attributes to images for accessibility and SEO.

Example of good SEO practices in HTML:
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="description" content="Learn the basics of SEO with HTML.">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>SEO Basics</title>
</head>
<body>
  <header>
    <h1>SEO Basics</h1>
  </header>
  <main>
    <section>
      <h2>What is SEO?</h2>
      <p>SEO stands for Search Engine Optimization...</p>
    </section>
    <section>
      <h2>How to do SEO with HTML?</h2>
      <p>Use semantic HTML elements, meta descriptions, proper headings, and alt attributes...</p>
    </section>
  </main>
  <footer>
    <p>&copy; 2024 SEO Basics</p>
  </footer>
</body>
``` 
This structure improves the readability of the content for your users and search engines.
