# Tables

Tables display tabular data using `<table>`. Rows are created with `<tr>`, header cells with `<th>`, and data cells with `<td>`.

---

Example of a simple table:

```html
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Charlie</td>
    <td>35</td>
  </tr>
  <tr>
    <td>Diana</td>
    <td>28</td>
  </tr>
</table>
```

The code will have an output like this:

| Name    | Age |
| ------- | --- |
| Alice   | 30  |
| Bob     | 25  |
| Charlie | 35  |
| Diana   | 28  |
