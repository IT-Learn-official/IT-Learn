# What is HTML?

HTML stands for **HyperText Markup Language.** Think of a web page as a house: HTML is the frame and rooms that define where things go: the walls, doors, and windows. It tells the browser what pieces of content exist and how they are nested.

---

_For example: when you write a recipe, you list ingredients and steps. HTML is like that list: it names the parts of the page (headings, paragraphs, images), not how they should look. If HTML is the ingredients, CSS is the recipe's presentation (how to plate it) and JavaScript is the chef who adds finishing touches and interaction._

---

## What HTML does and doesn't do:

- It structures content: headings, paragraphs, lists, links, images, forms.
- It does NOT define styling (colors, fonts, layout) — that's CSS.
- It does NOT run logic or change itself dynamically — that's JavaScript.

---

HTML vs CSS vs JavaScript: HTML provides structure, CSS provides presentation (styles), and JavaScript provides behavior and interactivity.
