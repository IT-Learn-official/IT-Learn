# Form Inputs Overview

This is a quick overview of all form input types that exist in HTML.

## Text inputs

- `text`: Single-line text field (default).
- `password`: Text field that masks characters.
- `email`: Email address with validation support.
- `tel`: Telephone number input.
- `url`: URL input with validation.
- `search`: Search query field, often with styling like a search icon.
- `number`: Numeric input with spinner controls.

## Selectable inputs

- `radio`: Single-choice button from a group.
- `checkbox`: Multiple-choice toggle.

## Button inputs

- `submit`: Form submission button.
- `button`: Clickable button for custom actions.
- `reset`: Clears form to defaults.
- `image`: Image acting as submit button.

## Date & time inputs

- `date`: Date picker (YYYY-MM-DD).
- `time`: Time picker (HH:MM).
- `datetime-local`: Local date and time.
- `month`: Month and year picker.
- `week`: Week picker.

## Other inputs

- `file`: File upload selector.
- `hidden`: Hidden form data.
- `color`: Color picker.
- `range`: Slider for numeric range.
