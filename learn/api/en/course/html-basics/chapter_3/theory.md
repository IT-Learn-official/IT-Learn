# HTML Document Structure

Every HTML page follows a predictable skeleton. It **starts** with `<!DOCTYPE html>` — this tells the browser we are using modern HTML. The top-level element is `<html>` which contains **two main parts:**

- `<head>`: Think of this as the envelope and labels for your page — it contains the page title, character settings, and links to files like CSS. These items **don't appear as page content** but they guide how the page is handled.
- `<body>`: This is the living room of the house where people see **content**: headings, paragraphs, images, lists, and buttons.

---

**For example:** imagine writing a letter. The `<head>` is the letterhead (who it's from and how to read it), and the `<body>` is the message itself.
