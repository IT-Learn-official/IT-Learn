# Adding Backgrounds

Backgrounds help sections feel different from each other.

You can use a plain color, a gradient, or an image.

```css
.hero {
  background: linear-gradient(90deg, #4facfe, #00f2fe);
}

.banner {
  background-image: url("stars.jpg");
  background-size: cover;
  background-position: center;
}
```

`cover` makes the image fill the area.
