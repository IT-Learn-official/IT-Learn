# Colors and Units

CSS gives you different ways to write colors.
You can use a name like `red`, a HEX code like `#ff0000`, RGB, or HSL.
They all work.

Sizes also have different units.
`px` is fixed.
`%` is relative.
`em` and `rem` are based on text size.

```css
.box {
  background-color: #4caf50;
  width: 50%;
  font-size: 1rem;
}
```

You do not need to master every unit on day one.
Start with `px`, `%`, and `rem`.
