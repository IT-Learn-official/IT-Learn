# Display and Positioning

`display` controls how an element behaves in the layout.
`position` controls where it sits.

Common display values: `block`, `inline`, `inline-block`.
Common position values: `static`, `relative`, `absolute`, `fixed`.

```css
.card {
  position: relative;
}

.badge {
  position: absolute;
  top: 8px;
  right: 8px;
}
```

This puts the badge at the top-right corner of the card.
