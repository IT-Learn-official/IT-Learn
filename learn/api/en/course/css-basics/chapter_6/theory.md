# Text Alignment and Spacing

Sometimes text feels "off" even when spelling is correct.
Usually, spacing is the problem.

`line-height` adds space between lines.
`text-align` moves text left, center, or right.
`letter-spacing` adjusts spacing between letters.

```css
p {
  line-height: 1.6;
  letter-spacing: 0.2px;
}

h2 {
  text-align: center;
}
```

Even tiny spacing changes can make reading much easier.
