# Flexbox for Beginners

Flexbox is a simple way to line up items.

First, turn on flexbox on the parent:

```css
.container {
  display: flex;
}
```

Then use properties like `justify-content`, `align-items`, and `gap`.

```css
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 12px;
}
```

It is one of the most useful layout tools in CSS.
