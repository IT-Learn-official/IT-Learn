# Basic Text Styling

Text is the first thing people read, so styling it well matters.

The most common text settings are:

- `font-family`
- `font-size`
- `font-weight`
- `color`

```css
body {
  font-family: Arial, sans-serif;
  font-size: 16px;
  color: #222;
}

h1 {
  font-weight: 700;
}
```

That last line makes the heading bold.
