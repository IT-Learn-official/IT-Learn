# Understanding Selectors and Properties

A CSS rule looks like this:

```css
selector {
  property: value;
}
```

Easy meanings:

- selector = what you pick
- property = what you change
- value = new setting
- declaration = one line (example: `color: blue;`)

Example:

```css
h1 {
  color: navy;
  font-size: 30px;
}
```

Here:

- h1 picks all h1 titles
- color changes text color
- font-size changes text size
