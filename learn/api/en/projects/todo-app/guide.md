# To-Do App (Vanilla JS)

Build a small to-do app. Focus on clear code and tiny steps.

## Requirements

- Add a new item
- Mark item as done/undone
- Remove an item
- Persist items in `localStorage`

## Steps

- [ ] Create the UI layout in `index.html` (input + button + list).
- [ ] Implement `addTodo()` in `app.js`.
- [ ] Implement toggling done state.
- [ ] Implement remove.
- [ ] Add persistence (`loadTodos()` + `saveTodos()`).
- [ ] Add small UX improvements (empty state, focus, keyboard enter).
