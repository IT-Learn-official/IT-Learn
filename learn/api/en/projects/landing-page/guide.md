# Responsive Landing Page

Build a simple landing page from scratch. Don’t copy/paste a template, try to implement each step yourself.

## Goal

- A header with navigation
- A hero section (headline, short text, call to action button)
- A features section (3 cards)
- A footer

## Steps

- [ ] Create the basic HTML structure in `index.html`.
- [ ] Add a header with a logo + 3 nav links.
- [ ] Build the hero section using a two-column layout.
- [ ] Create 3 feature cards using CSS grid.
- [ ] Make it responsive (mobile nav stacks, hero becomes one column).
- [ ] Polish: spacing, font sizes, hover states.

## Hints

- Use `max-width` + `margin: 0 auto` for a centered container.
- Use `@media (max-width: 800px)` for your breakpoint.
