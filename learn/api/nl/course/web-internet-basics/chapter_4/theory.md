# Hoofdstuk 4: Servers (De machines die jou antwoorden)

Servers zijn machines of software die verzoeken van browsers en apps beantwoorden.

Aan het einde van dit hoofdstuk kun je:

- Definiëren wat een server is.
- Uitleggen wat een webserver doet.
- Beschrijven hoe backend‑apps en databases samenwerken.

---

## Wat is een server?

Een server is een machine of dienst die luistert naar verzoeken en antwoorden verzendt. "Server" kan verwijzen naar de fysieke/virtuele machine of naar de software die erop draait (bv. Nginx, Apache, Node).

> **Belangrijk:** Een server luistert en antwoordt — hij levert de data of dienst die clients opvragen.

---

## Webserver vs Backend‑app

- **Webserver:** levert statische bestanden, handelt TLS af en stuurt vaak verzoeken door naar het backend.
- **Backend‑app:** voert applicatielogica uit (login, database‑toegang, API’s) en maakt dynamische antwoorden.

Moderne sites combineren client (browser), webserver (voordeur), backend‑app (logica) en database (opslag).
