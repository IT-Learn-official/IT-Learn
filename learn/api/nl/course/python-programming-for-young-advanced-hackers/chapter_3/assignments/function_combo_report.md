# Oefening: Functie-combo mini-rapport

Doel
- Combineer meerdere ingebouwde functies in één mini "status report" met `print(...)`.

Stappen
1. Open `attachments/function_combo_report.py`.
2. Gebruik deze functies elk minstens 1 keer:
   - `len`
   - `abs`
   - `round`
   - `type`
3. Print een rapport met meerdere regels (naam, lengte, afstand, afgeronde waarde, type, ...).
4. Voeg gerust extra variabelen en extra regels toe.

Checklist
- [ ] `len`, `abs`, `round` en `type` allemaal gebruikt.
- [ ] Duidelijk multi-line rapport geprint.
- [ ] Script draait zonder errors.
