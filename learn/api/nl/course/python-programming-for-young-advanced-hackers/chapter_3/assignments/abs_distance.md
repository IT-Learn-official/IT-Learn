# Oefening: Afstand tot nul met abs

Doel

- Gebruik `abs(...)` om de afstand van getallen tot 0 te berekenen (altijd 0 of positief).

Stappen

1. Open `attachments/abs_distance.py`.
2. Gebruik `abs(...)` minstens 2 keer: één keer op een negatief getal en één keer op een positief getal.
3. Bewaar elke uitkomst in een variabele (bv. `distance1 = abs(step1)`).
4. Print per getal een duidelijke zin met het originele getal en de afstand.

Checklist

- [ ] `abs(...)` minstens 2 keer gebruikt.
- [ ] Elke uitkomst in een variabele opgeslagen.
- [ ] Duidelijke output geprint met originele waarde + afstand.
- [ ] Script draait zonder errors.
