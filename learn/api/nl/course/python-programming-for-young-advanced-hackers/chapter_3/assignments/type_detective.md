# Oefening: Type-detective met type

Doel
- Gebruik `type(...)` om te zien welk type een variabele heeft (bv. `int`, `float`, `str`).

Stappen
1. Open `attachments/type_detective.py`.
2. Gebruik `type(...)` op minstens 3 verschillende variabelen.
3. Print per variabele een duidelijke zin, bijvoorbeeld: `coins is of type <...>`.
4. (Extra) Probeer het type te raden voordat je runt, en check of je gelijk had.

Checklist
- [ ] `type(...)` gebruikt op minstens 3 variabelen.
- [ ] Type-informatie geprint met duidelijke boodschappen.
- [ ] Je kan het verschil uitleggen tussen `int`, `float` en `str`.
