# Oefening: Scorebord afronden met round

Doel
- Gebruik `round(...)` om "rommelige" decimalen leesbaar te maken.

Stappen
1. Open `attachments/round_scoreboard.py`.
2. Rond minstens één getal af naar een geheel getal (één argument: `round(x)`).
3. Rond minstens één ander getal af naar 2 decimalen (twee argumenten: `round(x, 2)`).
4. Bewaar de afgeronde waarden in variabelen.
5. Print een klein scorebord met de originele en afgeronde waarden.

Checklist
- [ ] `round(x)` gebruikt (1 argument).
- [ ] `round(x, n)` gebruikt (2 argumenten).
- [ ] Resultaten opgeslagen in variabelen.
- [ ] Duidelijke scoreboard-output geprint.
