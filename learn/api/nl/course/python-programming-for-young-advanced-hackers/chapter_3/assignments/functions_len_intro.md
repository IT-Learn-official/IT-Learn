# Oefening: Tekens tellen met len

Doel
- Gebruik `len(...)` om de lengte van strings te meten, sla de resultaten op in variabelen en print ze duidelijk.

Stappen
1. Open `attachments/functions_len_intro.py`.
2. Roep `len(...)` minstens 3 keer aan op verschillende strings (bv. `"cat"`, `"dragon"`, `"Hello, world!"`).
3. Bewaar elke uitkomst in een variabele (bv. `length_cat = len("cat")`).
4. Print per string een zin die duidelijk toont wat je gemeten hebt.

Checklist
- [ ] `len(...)` minstens 3 keer gebruikt.
- [ ] Elke uitkomst in een variabele opgeslagen.
- [ ] Duidelijke `print(...)`-regels toegevoegd.
- [ ] Script draait zonder errors en toont de juiste lengtes.
