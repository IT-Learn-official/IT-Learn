# Missie: Message analyzer

Doel
- Analyseer een geheim bericht met indexing, slicing en hoofd-/kleine letters.

Stappen
1. Open `attachments/message_analyzer.py`.
2. Start met een `message` string in je code (bv. `"Top Secret: HACK THE PLANET"`).
3. Print:
   - het **eerste** teken (`message[0]`)
   - het **laatste** teken (`message[-1]`)
   - een slice die het geheime woord bevat (`message[start:end]`)
4. Print het volledige bericht in `lowercase` en daarna in `UPPERCASE` met `.lower()` en `.upper()`.
