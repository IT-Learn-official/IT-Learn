# Missie: Cleaner bot

Doel

- Maak een bot die rommelige tekst opschoont met string-methodes.

Stappen

1. Open `attachments/cleaner_bot.py`.
2. Start met voorbereid tekst met extra spaties / ongewenste woorden.
3. Gebruik:
   - `.strip()` om spaties voor/achteraan te verwijderen
   - `.replace(old, new)` om woorden te verbergen of te wijzigen
   - `.split(",")` op comma-separated tekst en print elk item op een nieuwe lijn
