# Missie: Secret name tag

Doel
- Maak een geheime hackernaam op basis van voornaam en achternaam met slicing en een f-string.

Stappen
1. Open `attachments/secret_name_tag.py`.
2. Maak `first_name` en `last_name`.
3. Gebruik slicing om een stuk van elke naam te nemen (bv. de eerste 3 letters).
4. Combineer die stukken tot `code_name`.
5. Print het resultaat met een f-string, bv. `Your hacker name is: ...`.
