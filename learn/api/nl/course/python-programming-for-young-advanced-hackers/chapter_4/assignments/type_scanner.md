# Missie: Type scanner

Doel
- Maak variabelen met verschillende datatypes en print telkens de waarde én `type(...)`.

Stappen
1. Open `attachments/type_scanner.py`.
2. Maak minstens 4 variabelen met verschillende types:
   - 1 `int` (bv. `42`)
   - 1 `float` (bv. `3.14`)
   - 1 `str` die op een getal lijkt (bv. `"123"`)
   - 1 `bool` (bv. `True`)
3. Print per variabele de waarde en het type met `type()`.

Checklist
- [ ] Minstens 4 variabelen, allemaal van een ander type.
- [ ] Minstens 4 lijnen output met waarde + `type(...)`.
- [ ] Script draait zonder errors.
