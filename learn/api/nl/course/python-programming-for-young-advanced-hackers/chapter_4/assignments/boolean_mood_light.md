# Missie: Boolean mood light

Doel
- Toon met `bool(...)` welke waarden als True/False tellen.

Stappen
1. Open `attachments/boolean_mood_light.py`.
2. Maak of gebruik minstens 4 variabelen met verschillende waarden (bv. `True`, `""`, `0`, `"hi"`).
3. Print per variabele:
   - de naam
   - de waarde
   - `bool(waarde)`
4. Voeg minstens 1 extra variabele van jezelf toe en print die ook.

Tip
- Gebruik `repr(...)` voor strings als je leeg vs. niet-leeg duidelijk wil zien.

Checklist
- [ ] Minstens 4 lijnen output.
- [ ] Elke lijn toont een waarde én `bool(...)`.
- [ ] Script draait zonder errors.
