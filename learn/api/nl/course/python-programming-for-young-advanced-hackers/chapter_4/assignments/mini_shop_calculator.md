# Missie: Mini shop calculator

Doel

- Bouw een mini shop-calculator die prijs en hoeveelheid (als strings) omzet naar getallen en een subtotaal berekent.

Stappen

1. Open `attachments/mini_shop_calculator.py`.
2. Zet `price_str` om naar een `float` met `float()`.
3. Zet `quantity_str` om naar een `int` met `int()`.
4. Bereken `subtotal = price * quantity`.
5. Print het subtotaal duidelijk (bv. `Subtotal: 29.97`).

Optioneel

- Voeg `discount_active = True/False` toe en print een extra lijn als de korting actief is.

Checklist

- [ ] `float()` gebruikt voor de prijs.
- [ ] `int()` gebruikt voor de hoeveelheid.
- [ ] Subtotaal correct berekend en geprint.
- [ ] Script draait zonder errors.
