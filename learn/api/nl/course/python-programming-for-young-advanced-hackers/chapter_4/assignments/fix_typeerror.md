# Missie: Fix de TypeError

Doel

- Fix de starter code in `attachments/fix_typeerror.py` die strings en getallen door elkaar gebruikt en daardoor crasht met `TypeError`.

Stappen

1. Open `attachments/fix_typeerror.py`.
2. Zet nummer-achtige strings om naar echte getallen vóór je ermee rekent (met `int()` of `float()` waar nodig).
3. Als je tekst en getallen samen wil printen:
   - gebruik `str(...)`, of
   - gebruik komma's in `print(...)`.
4. Verander het verhaal of de getallen niet; fix alleen de types.

Checklist

- [ ] Script draait zonder errors.
- [ ] Output is logisch (bv. 16, 150, 15 zoals in de opdracht bedoeld).
- [ ] Je gebruikt `int()`/`float()` voor rekenen en `str()`/komma's voor tekst + getal.
