# Missie: Logische operatoren challenge

Doel

- Probeer de logische operatoren: `and`, `or`, `not`.

Stappen

1. Open `attachments/logical_operators.py`.
2. Gebruik elke logische operator in booleaanse expressies en print het resultaat.
3. Zorg voor minstens 3 `print(...)`-regels (één per operator).
