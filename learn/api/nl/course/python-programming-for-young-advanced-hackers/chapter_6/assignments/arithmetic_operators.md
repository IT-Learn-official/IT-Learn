# Missie: Rekenoperatoren playground

Doel

- Probeer alle rekenoperatoren: `+`, `-`, `*`, `/`, `//`, `%` en `**`.

Stappen

1. Open `attachments/arithmetic_operators.py`.
2. Gebruik elke operator op minstens twee getallen en print het resultaat.
3. Zorg voor minstens 6 `print(...)`-regels (één per operator).
