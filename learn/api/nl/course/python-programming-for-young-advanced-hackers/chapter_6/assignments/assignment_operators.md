# Missie: Toekenningsoperatoren lab

Doel

- Oefen met toekenningsoperatoren: `=`, `+=`, `-=`, `*=`, `/=`.

Stappen

1. Open `attachments/assignment_operators.py`.
2. Start met één variabele en update die met elke operator.
3. Print de waarde na elke wijziging.
4. Zorg voor minstens 5 `print(...)`-regels (één per operator).
