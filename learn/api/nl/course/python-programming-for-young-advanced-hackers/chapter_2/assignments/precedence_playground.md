# Missie: Haakjes power-up

Doel
- Toon dat haakjes `()` de volgorde van bewerkingen veranderen.

Stappen
1. Open `attachments/precedence_playground.py`.
2. Voeg printregels toe met paren expressies: één zonder haakjes en één met haakjes.
3. Voeg minstens 2 extra paren toe, waarvan minstens 1 met `//` (gehele deling) of `%` (rest).

Checklist
- [ ] Minstens 2 paren expressies toegevoegd.
- [ ] Minstens 2 extra paren toegevoegd.
- [ ] Minstens 1 voorbeeld gebruikt `//` of `%`.
- [ ] Script draait zonder errors.
