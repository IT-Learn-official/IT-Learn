# Missie: Verdeel de buit

Doel
- Bereken hoeveel munten elke speler krijgt met `//` en hoeveel er overblijft met `%`.

Stappen
1. Open `attachments/share_the_loot.py`.
2. Stel `total_coins` en `players` in.
3. Bereken:
   - `coins_per_player = total_coins // players`
   - `leftover_coins = total_coins % players`
4. Print een korte, leesbare samenvatting.

Checklist
- [ ] `//` gebruikt voor `coins_per_player`.
- [ ] `%` gebruikt voor `leftover_coins`.
- [ ] Resultaten worden geprint in duidelijke zinnen.
- [ ] Script draait zonder errors.
