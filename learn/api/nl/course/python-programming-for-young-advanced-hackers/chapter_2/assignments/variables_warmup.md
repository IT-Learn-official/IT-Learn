# Missie: Geef je geheime getallen een naam

Doel

- Maak minstens 3 numerieke variabelen (minstens 1 `int` en 1 `float`) en print ze met een label.

Stappen

1. Open `attachments/variables_warmup.py`.
2. Maak minstens drie variabelen, bijvoorbeeld `lives = 3`, `speed = 1.5`, `coins = 10`.
3. Print elke variabele netjes met een label (met komma's in `print(...)` of met `str(...)`).

Checklist

- [ ] Minstens 3 variabelen aangemaakt.
- [ ] Minstens 1 float gebruikt (met een punt, bv. `1.5`).
- [ ] Elke variabele wordt geprint met een label.
- [ ] Script draait zonder errors.
