# Missie: Rounding gym - round, floor en ceil

Doel
- Train afronden met `round`, `math.floor` en `math.ceil`.

Stappen
1. Open `attachments/rounding_gym.py`.
2. Vraag een decimaal getal aan de gebruiker (bv. `3.7` of `-1.2`).
3. Print duidelijk:
   - Rounded: `round(x)`
   - Floor: `math.floor(x)`
   - Ceil: `math.ceil(x)`

Hints
- `import math`
- floor gaat altijd naar beneden, ceil altijd naar boven.
