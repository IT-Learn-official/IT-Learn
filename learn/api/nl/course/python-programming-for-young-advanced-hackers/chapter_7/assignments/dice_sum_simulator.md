# Missie: Dobbelsteen - som van veel worpen

Doel

- Simuleer het rollen van een 6-zijdige dobbelsteen en tel alle worpen op.

Stappen

1. Open `attachments/dice_sum_simulator.py`.
2. Vraag aan de gebruiker hoe vaak er gerold moet worden.
3. Gebruik `random.randint(1, 6)` voor elke worp.
4. Print (optioneel) elke worp als `Roll 1: 4`.
5. Hou een `total` bij en print op het einde de totaalsom.

Hints

- `import random`
- Start met `total = 0` vóór je loop.
