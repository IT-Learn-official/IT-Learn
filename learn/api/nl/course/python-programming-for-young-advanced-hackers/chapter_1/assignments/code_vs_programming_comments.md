# Markeer planning vs code

Taak
- Voeg inline markers toe in `attachments/code_vs_programming_comments.py`.

Regels
- Zet `# PROGRAMMING` op het einde van elke planning/comment‑regel.
- Zet `# CODE` op het einde van elke uitvoerbare Python‑regel.
- Verwijder of herschik geen regels.
- Het bestand moet blijven runnen zonder syntax errors.

Validator hint
- De validator verwacht dat `name = getName(` en `print("Hello, " + name + "!")` aanwezig zijn met de juiste markers.
