# Missie: Trace de score

Doel
- Trace hoe `score` verandert door `print(...)` toe te voegen na elke update.

Stappen
1. Open `attachments/trace_the_score.py`.
2. Voeg na elke wijziging van de score een `print(...)` toe.
3. Run de code en zorg dat je output exact dit is:

```
Score: 0
Score: 10
Score: 15
```
