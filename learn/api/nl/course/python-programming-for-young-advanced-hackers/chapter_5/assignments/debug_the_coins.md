# Missie: Debug de coins

Doel

- Gebruik print-debugging om de waarde van `coins` te tracen en de updates te fixen.

Stappen

1. Open `attachments/debug_the_coins.py`.
2. Voeg `print(...)`-regels toe na elke verandering van `coins`.
3. Fix de fouten zodat de output exact dit is:

```
Start coins: 5
After buying a shield: 3
After finding treasure: 8
```
