# Schaduwen en randen

Randen, afgeronde hoeken en schaduwen kunnen een ontwerp netter en strakker laten aanvoelen.

```css
.card {
  border: 1px solid #ddd;
  border-radius: 12px;
  box-shadow: 0 8px 20px rgba(0, 0, 0, 0.12);
}
```

Je hoeft geen zware schaduwen te gebruiken.
Kleine schaduwen zien er vaak beter uit.
