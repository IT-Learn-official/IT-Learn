# Kleuren en eenheden

CSS geeft je verschillende manieren om kleuren te schrijven.
Je kan een naam gebruiken zoals `red`, een HEX-code zoals `#ff0000`, RGB of HSL.
Het werkt allemaal.

Groottes hebben ook verschillende eenheden.
`px` is vast.
`%` is relatief.
`em` en `rem` zijn gebaseerd op tekengrootte.

```css
.box {
  background-color: #4caf50;
  width: 50%;
  font-size: 1rem;
}
```

Je moet niet elke eenheid op dag één beheersen.
Begin met `px`, `%` en `rem`.
