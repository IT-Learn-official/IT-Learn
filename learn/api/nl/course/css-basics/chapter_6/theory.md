# Tekstuitlijning en spatiëring

Soms voelt tekst “net niet goed”, zelfs als de spelling klopt.
Meestal is spatiëring het probleem.

`line-height` voegt ruimte toe tussen regels.
`text-align` zet tekst links, gecentreerd of rechts.
`letter-spacing` past de ruimte tussen letters aan.

```css
p {
  line-height: 1.6;
  letter-spacing: 0.2px;
}

h2 {
  text-align: center;
}
```

Zelfs kleine wijzigingen in spatiëring maken lezen vaak een pak makkelijker.
