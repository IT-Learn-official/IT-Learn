# Selectors en properties begrijpen

Een CSS-regel ziet er zo uit:

```css
selector {
  property: value;
}
```

Eenvoudig uitgelegd:
- selector = wat je selecteert
- property = wat je verandert
- value = de nieuwe instelling
- declaration = één regel (voorbeeld: `color: blue;`)

Voorbeeld:

```css
h1 {
  color: navy;
  font-size: 30px;
}
```

Hier:
- h1 selecteert alle h1-titels
- color verandert de tekstkleur
- font-size verandert de tekstgrootte
