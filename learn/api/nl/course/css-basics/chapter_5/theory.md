# Basis tekststijl

Tekst is het eerste wat mensen lezen, dus het is belangrijk dat je die netjes stylet.

De meest voorkomende tekstinstellingen zijn:
- `font-family`
- `font-size`
- `font-weight`
- `color`

```css
body {
  font-family: Arial, sans-serif;
  font-size: 16px;
  color: #222;
}

h1 {
  font-weight: 700;
}
```

Die laatste regel maakt de titel vet.
