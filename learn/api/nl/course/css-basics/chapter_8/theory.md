# Het CSS box model uitgelegd

In CSS wordt elk element behandeld alsof het een doos (box) is.

Die box heeft vier delen:
content, padding, border en margin.

```css
.card {
  padding: 16px;
  border: 2px solid #333;
  margin: 20px;
}
```

Denk aan een schilderij in een kader. Je kan het opdelen in 4 stukken:

- Het schilderij zelf (de content)
- De witruimte tussen schilderij en kader (de padding)
- Het kader (de border)
- De muur errond (de margin)
