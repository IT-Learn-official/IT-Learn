# Wat is CSS en waarom gebruik je het?

Als je een gewone HTML-pagina opent, werkt die, maar meestal ziet ze er wat saai uit.
Daar komt CSS van pas.

CSS is het deel dat je pagina stijl geeft.
Het verandert kleuren, tekengroottes, witruimte en layout.

Dus: HTML is de structuur, CSS is het ontwerp.

Als HTML nog nieuw aanvoelt, begin dan eerst hier:
https://itlearn.be/learn/#/courses?courseId=html-basics

---

Klein voorbeeld:

```html
<h1 class="title">Hello</h1>
```

```css
.title {
  color: blue;
}
```

Die ene CSS-regel maakt de titel blauw.
