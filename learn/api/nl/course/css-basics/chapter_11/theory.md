# Display en positionering

`display` bepaalt hoe een element zich gedraagt in de layout.
`position` bepaalt waar het precies komt te staan.

Veelgebruikte display-waarden: `block`, `inline`, `inline-block`.
Veelgebruikte position-waarden: `static`, `relative`, `absolute`, `fixed`.

```css
.card {
  position: relative;
}

.badge {
  position: absolute;
  top: 8px;
  right: 8px;
}
```

Zo staat de badge in de rechterbovenhoek van de card.
