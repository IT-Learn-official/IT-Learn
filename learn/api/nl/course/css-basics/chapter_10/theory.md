# Achtergronden toevoegen

Achtergronden helpen om secties duidelijk van elkaar te onderscheiden.

Je kan een effen kleur gebruiken, een gradient, of een afbeelding.

```css
.hero {
  background: linear-gradient(90deg, #4facfe, #00f2fe);
}

.banner {
  background-image: url("stars.jpg");
  background-size: cover;
  background-position: center;
}
```

`cover` zorgt ervoor dat de afbeelding het hele vlak opvult.
