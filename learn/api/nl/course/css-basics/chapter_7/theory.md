# Links stylen

Links kunnen er anders uitzien afhankelijk van wat de gebruiker doet.

`a` is de normale toestand.
`a:hover` is wanneer je met de muis over de link gaat.
`a:visited` is een link die de gebruiker al geopend heeft.

```css
a {
  color: #0b63f6;
  text-decoration: none;
}

a:hover {
  color: #0847b8;
  text-decoration: underline;
}
```

Zo voelen links meer interactief aan.
