# HTML + CSS

CSS staat voor Cascading Style Sheets. Het is een taal waarmee je je website een eigen look & feel geeft. HTML bepaalt de structuur van je webpagina, CSS bepaalt hoe die structuur aan gebruikers gepresenteerd wordt. Met CSS controleer je kleuren, lettertypes, spatiëring, layout, en nog veel meer.

---

Denk aan CSS alsof je een huis inricht: HTML is het geraamte en de kamers, terwijl CSS de verf, meubels en decoratie is die het er mooi laten uitzien. (In deze vergelijking is JavaScript dan de elektriciteit en het sanitair: die zorgen voor interactiviteit en functionaliteit.)
