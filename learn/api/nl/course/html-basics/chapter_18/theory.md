# SEO basics

SEO staat voor Search Engine Optimization. Simpel gezegd betekent het dat je je website makkelijker “leesbaar” maakt voor zoekmachines zoals Google en Bing. Daardoor kan je website hoger ranken in zoekresultaten, en kunnen meer mensen je site vinden.

---

SEO doen met HTML:
- Gebruik semantische HTML-elementen zoals `<header>`, `<nav>`, `<main>`, `<section>` en `<footer>` om je content te structureren.
- Voeg een duidelijke meta description toe in de `<head>` van je HTML.
- Gebruik heading tags (`<h1>` tot `<h6>`) correct om hiërarchie aan te geven.
- Voeg alt-attributen toe aan afbeeldingen voor toegankelijkheid en SEO.

Voorbeeld van goede SEO-praktijken in HTML:
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="description" content="Learn the basics of SEO with HTML.">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>SEO Basics</title>
</head>
<body>
  <header>
    <h1>SEO Basics</h1>
  </header>
  <main>
    <section>
      <h2>What is SEO?</h2>
      <p>SEO stands for Search Engine Optimization...</p>
    </section>
    <section>
      <h2>How to do SEO with HTML?</h2>
      <p>Use semantic HTML elements, meta descriptions, proper headings, and alt attributes...</p>
    </section>
  </main>
  <footer>
    <p>&copy; 2024 SEO Basics</p>
  </footer>
</body>
```
Deze structuur maakt de inhoud beter leesbaar voor je gebruikers én voor zoekmachines.
