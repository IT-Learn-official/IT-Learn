# Backend vs frontend

Je hebt de termen backend en frontend waarschijnlijk al eens gehoord. Maar wat betekenen ze?

Backend:

- is het deel van de website dat gebruikers niet zien.
- behandelt data-opslag, serverlogica en de functionaliteit van de applicatie.
- voorbeelden: databanken, server-side programmeertalen (zoals Python, Node.js), API’s.

Frontend:

- is het deel waar gebruikers mee interageren.
- behandelt de visuele layout, het design en de gebruikerservaring.
- voorbeelden: HTML, CSS, JavaScript.
