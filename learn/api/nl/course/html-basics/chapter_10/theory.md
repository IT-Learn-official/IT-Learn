# Tabellen

Tabellen tonen gegevens in rijen en kolommen met `<table>`. Rijen maak je met `<tr>`, header-cellen met `<th>` en datacellen met `<td>`.

---

Voorbeeld van een eenvoudige tabel:

```html
<table>
  <tr>
    <th>Naam</th>
    <th>Leeftijd</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Charlie</td>
    <td>35</td>
  </tr>
  <tr>
    <td>Diana</td>
    <td>28</td>
  </tr>
</table>
```

De output ziet er ongeveer zo uit:

| Naam    | Leeftijd |
| ------- | -------- |
| Alice   | 30       |
| Bob     | 25       |
| Charlie | 35       |
| Diana   | 28       |
