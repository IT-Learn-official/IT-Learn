# Lijsten

Lijsten organiseren items die bij elkaar horen. Gebruik `<ul>` voor ongeordende (met bolletjes) lijsten en `<ol>` voor geordende (genummerde) lijsten. Elk item komt in een `<li>`-element.

---

Voorbeeld van een ongeordende lijst:

```html
<ul>
  <li>Melk</li>
  <li>Brood</li>
  <li>Eieren</li>
</ul>
```

Dit ziet er zo uit:

- Melk
- Brood
- Eieren

---

Voorbeeld van een geordende lijst:

```html
<ol>
  <li>Verwarm de oven voor op 350°F.</li>
  <li>Meng de ingrediënten.</li>
  <li>Bak 30 minuten.</li>
</ol>
```

Dit wordt getoond als:

1. Verwarm de oven voor op 350°F.
2. Meng de ingrediënten.
3. Bak 30 minuten.
