# Paragrafen & tekst

Tekst is de meest voorkomende content op websites. Met `<p>` maak je paragrafen in HTML. Voor een enkele regelafbreking gebruik je `<br>`, en voor een thematische scheiding gebruik je `<hr>`.

Tags voor nadruk:
- `<strong>` markeert tekst als extra belangrijk. Browsers tonen dit meestal **vet**.
- `<em>` zet tekst in de verf als nadruk. Browsers tonen dit meestal *cursief*.

Voorbeeld:
```html
<p>This is a paragraph of text.</p>
<hr>
<p>This is another paragraph with a <strong>bold</strong> word
and an <em>italic</em> word.</p>
```

Output:
>This is a paragraph of text.
>
>---
>This is another paragraph with a **bold** word and an *italic* word.
