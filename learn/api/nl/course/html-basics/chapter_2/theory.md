# Setup: vereisten

Om HTML te schrijven heb je maar twee tools nodig: een teksteditor (zoals VS Code) en een moderne webbrowser (Chrome, Firefox, Edge). In de editor typ je je pagina; de browser leest je bestand en toont het alsof het een afgewerkte webpagina is.

---

Alledaags voorbeeld: denk aan een potlood en een schetsboek. Het potlood is je editor en het schetsboek is je browser preview – samen kun je snel ideeën uitschrijven en bekijken.

---

## Snelle stappen om je eerste HTML-bestand te maken:
- Open je editor en maak een bestand met de naam `index.html`.
- Zet er een basis HTML-skelet in en voeg een kleine koptekst toe.
- Sla op en open het bestand in je browser om het resultaat te zien.

Hier is een basis HTML-template om te starten:

```html
<!DOCTYPE html>
<html lang="nl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mijn eerste HTML-pagina</title>
</head>
<body>
    <h1>Hallo wereld!</h1>
</body>
</html>
```
