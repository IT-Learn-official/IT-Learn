# HTML-documentstructuur

Elke HTML-pagina volgt een herkenbaar skelet. Ze **start** met `<!DOCTYPE html>` – dat vertelt de browser dat we moderne HTML gebruiken. Het top-level element is `<html>` en dat bevat **twee hoofdonderdelen:**

- `<head>`: zie dit als de envelop en labels van je pagina – het bevat o.a. de paginatitel, karakterinstellingen en links naar bestanden zoals CSS. Deze dingen **verschijnen niet als zichtbare content**, maar sturen wel hoe de pagina verwerkt wordt.
- `<body>`: dit is de “woonkamer” waar mensen de **content** zien: kopteksten, paragrafen, afbeeldingen, lijsten en knoppen.

---

**Voorbeeld:** denk aan een brief. De `<head>` is de briefhoofding (van wie het is en hoe je het moet lezen) en de `<body>` is de boodschap zelf.
