# HTML comments

We gebruiken HTML comments om notities of uitleg toe te voegen in code. Die comments zijn niet zichtbaar voor de gebruiker. Comments schrijf je zo: `<!-- I am a comment -->`.

Je kan comments ook gebruiken om tijdelijk code uit te schakelen zonder ze te verwijderen. Bijvoorbeeld:
```html
<p>This paragraph will be displayed.</p>
<!-- <p>This paragraph is commented out and won't be displayed.</p> -->
```
