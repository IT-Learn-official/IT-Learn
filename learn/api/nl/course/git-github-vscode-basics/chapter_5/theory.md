# GitHub remotes: clone, push, pull

Je repo kan op twee plaatsen staan:

- lokaal (je computer)
- remote (GitHub)

Clone = een remote repo downloaden:

```bash
git clone <repo-url>
```

Push = je commits uploaden:

```bash
git push
```

Pull = nieuwe commits downloaden:

```bash
git pull
```

---

Na het clonen noemt Git de remote meestal `origin`.
Je kan dat checken met:

```bash
git remote -v
```
