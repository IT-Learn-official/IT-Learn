# To Do App (Vanilla JS)

Bouw een kleine to do app. Focus op duidelijke code en kleine stappen.

## Vereisten

- Item toevoegen
- Item done/undone togglen
- Item verwijderen
- Items opslaan in `localStorage`

## Stappen

- [ ] UI layout in `index.html` (input + knop + lijst).
- [ ] `addTodo()` in `app.js` implementeren.
- [ ] Done state togglen.
- [ ] Remove implementeren.
- [ ] Persistente opslag (`loadTodos()` + `saveTodos()`).
- [ ] UX verbeteren (empty state, focus, enter).
