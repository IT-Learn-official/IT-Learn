# Responsieve Landingpagina

Bouw een simpele landingpagina van nul. Niet copy/pasten, implementeer elke stap zelf.

## Doel

- Header met navigatie
- Hero (titel, korte tekst, call to action)
- Features (3 kaarten)
- Footer

## Stappen

- [ ] Basisstructuur in `index.html`.
- [ ] Header met logo + 3 nav links.
- [ ] Hero als 2 koloms layout.
- [ ] 3 feature cards met CSS grid.
- [ ] Responsief maken (mobile: alles onder elkaar).
- [ ] Afwerken: spacing, typografie, hover states.
