# To-Do-App (Vanilla-JS)

Baue eine kleine To-Do-App. Fokus: klare Struktur und kleine Schritte.

## Anforderungen

- Item hinzufügen
- Item erledigt/unerledigt toggeln
- Item entfernen
- Persistenz via `localStorage`
