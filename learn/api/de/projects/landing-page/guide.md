# Responsive Landingpage

Baue eine simple Landingpage von Grund auf. Nicht copy/pasten, implementiere jeden Schritt selbst.

## Ziel

- Header mit Navigation
- Hero Bereich (Headline, kurzer Text, Call to Action)
- Features Bereich (3 Karten)
- Footer

## Schritte

- [ ] Grundstruktur in `index.html` erstellen.
- [ ] Header mit Logo + 3 Nav Links hinzufügen.
- [ ] Hero Bereich als 2 Spalten Layout bauen.
- [ ] 3 Feature Cards per CSS Grid erstellen.
- [ ] Responsiv machen (Mobile: Navigation/Spalten stapeln).
- [ ] Feinschliff: Abstände, Typografie, Hover states.

## Tipps

- Nutze `max-width` + `margin: 0 auto` für einen Container.
- Nutze `@media (max-width: 800px)` als Breakpoint.
