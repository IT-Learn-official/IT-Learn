# Überblick über Form-Inputs

Hier ist ein schneller Überblick über die Form-Input-Typen in HTML.

## Text-Inputs

- `text`: einzeiliges Textfeld (Standard).
- `password`: Textfeld, das Zeichen maskiert.
- `email`: E-Mail-Adresse mit Validierungsunterstützung.
- `tel`: Eingabe für Telefonnummern.
- `url`: URL-Input mit Validierung.
- `search`: Suchfeld, oft mit Styling wie einem Such-Icon.
- `number`: numerische Eingabe mit Spinner-Controls.

## Auswahl-Inputs

- `radio`: eine Auswahl aus einer Gruppe.
- `checkbox`: mehrere Auswahlmöglichkeiten an/aus.

## Button-Inputs

- `submit`: Button zum Absenden des Formulars.
- `button`: klickbarer Button für eigene Aktionen.
- `reset`: setzt das Formular auf Default zurück.
- `image`: Bild als Submit-Button.

## Datum- & Zeit-Inputs

- `date`: Datums-Picker (YYYY-MM-DD).
- `time`: Zeit-Picker (HH:MM).
- `datetime-local`: lokales Datum und Uhrzeit.
- `month`: Monats- und Jahresauswahl.
- `week`: Wochenauswahl.

## Andere Inputs

- `file`: Dateiupload.
- `hidden`: versteckte Form-Daten.
- `color`: Farbwähler.
- `range`: Slider für einen Zahlenbereich.
