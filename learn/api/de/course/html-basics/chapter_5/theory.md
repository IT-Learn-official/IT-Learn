# Überschriften

Überschriften sind die großen Titel auf einer Seite. Sie reichen von `<h1>` (großer Titel) bis `<h6>` (kleinerer Untertitel).
Hier sind Beispiele, wie Überschriften aussehen:

# H1

## H2

### H3

#### H4

##### H5

###### H6

---

Fun Fact: Suchmaschinen und Screenreader nutzen Überschriften, um die Struktur deiner Seite zu verstehen.

Wichtige Regeln:

- Meistens reicht ein `<h1>` pro Seite (typischerweise der Haupttitel).
- Wähle Ebenen nach Struktur und semantischer Verschachtelung (`<h1>`, `<h2>`, `<h3>`, …), nicht nach optischer Größe.
- Trenne Struktur und Styling: nutze Überschriften für Bedeutung und CSS für das Aussehen.
- Mehr als ein `<h1>` ist nur dann sinnvoll, wenn es einen guten strukturellen Grund gibt.
