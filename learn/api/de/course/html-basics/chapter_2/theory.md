# Setup-Voraussetzungen

Um HTML zu schreiben, brauchst du nur zwei Tools: einen Text-Editor (z. B. VS Code) und einen modernen Webbrowser (Chrome, Firefox, Edge). Im Editor schreibst du deine Seite; der Browser liest die Datei und zeigt sie wie eine fertige Webseite an.

---

Alltagsvergleich: Stell dir einen Bleistift und ein Skizzenbuch vor. Der Bleistift ist dein Editor und das Skizzenbuch ist die Browser-Vorschau – zusammen kannst du Ideen schnell entwerfen und direkt ansehen.

---

## Schnelle Schritte für deine erste HTML-Datei:
- Öffne deinen Editor und erstelle eine Datei namens `index.html`.
- Füge das grundlegende HTML-Grundgerüst ein und eine kleine Überschrift.
- Speichere und öffne die Datei im Browser, um das Ergebnis zu sehen.

Hier ist eine einfache HTML-Vorlage zum Start:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My first html page</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>
```
