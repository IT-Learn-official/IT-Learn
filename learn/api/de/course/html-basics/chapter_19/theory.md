# HTML + CSS

CSS steht für Cascading Style Sheets. Damit gibst du deiner Website Look & Feel. Während HTML die Struktur deiner Seite definiert, legt CSS fest, wie diese Struktur für User aussieht. Mit CSS steuerst du Farben, Schriften, Abstände, Layout und vieles mehr.

---

Denk an CSS wie ans Hausbauen: HTML ist das Grundgerüst und die Räume, CSS sind Farbe, Möbel und Deko, die alles schön wirken lassen. (In dieser Analogie wäre JavaScript der Strom und die Leitungen – also Interaktivität und Funktion.)
