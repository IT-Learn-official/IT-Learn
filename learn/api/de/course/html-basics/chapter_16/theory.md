# HTML-Kommentare

HTML-Kommentare sind Notizen oder Erklärungen im Code. Sie sind für den User nicht sichtbar. Kommentare schreibst du so: `<!-- I am a comment -->`.

Du kannst Kommentare auch nutzen, um Code vorübergehend zu deaktivieren, ohne ihn zu löschen. Beispiel:

```html
<p>This paragraph will be displayed.</p>
<!-- <p>This paragraph is commented out and won't be displayed.</p> -->
```
