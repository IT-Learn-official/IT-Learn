# Absätze & Text

Text ist der häufigste Inhalt auf Websites. Mit `<p>` machst du Absätze in HTML. Für einen Zeilenumbruch nutzt du `<br>`, und für eine thematische Trennlinie `<hr>`.

Tags für Hervorhebung:

- `<strong>` markiert Text als besonders wichtig. Browser zeigen das meist **fett** an.
- `<em>` markiert Text mit Betonung. Browser zeigen das meist _kursiv_ an.

Beispiel:

```html
<p>This is a paragraph of text.</p>
<hr />
<p>
  This is another paragraph with a <strong>bold</strong> word and an
  <em>italic</em> word.
</p>
```

Ausgabe:

> This is a paragraph of text.
>
> ---
>
> This is another paragraph with a **bold** word and an _italic_ word.
