# SEO-Basics

SEO steht für Search Engine Optimization. Das bedeutet im Grunde: Deine Website für Suchmaschinen wie Google oder Bing leichter verständlich machen. Dadurch kann deine Website in Suchergebnissen höher ranken – und mehr Menschen finden dich.

---

SEO mit HTML:
- Nutze semantische HTML-Elemente wie `<header>`, `<nav>`, `<main>`, `<section>` und `<footer>`, um Inhalte zu strukturieren.
- Füge im `<head>` eine aussagekräftige Meta-Description hinzu.
- Verwende Überschriften (`<h1>` bis `<h6>`) korrekt, um die Hierarchie zu zeigen.
- Nutze alt-Attribute bei Bildern für Accessibility und SEO.

Beispiel für gute SEO-Praktiken in HTML:
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="description" content="Learn the basics of SEO with HTML.">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>SEO Basics</title>
</head>
<body>
  <header>
    <h1>SEO Basics</h1>
  </header>
  <main>
    <section>
      <h2>What is SEO?</h2>
      <p>SEO stands for Search Engine Optimization...</p>
    </section>
    <section>
      <h2>How to do SEO with HTML?</h2>
      <p>Use semantic HTML elements, meta descriptions, proper headings, and alt attributes...</p>
    </section>
  </main>
  <footer>
    <p>&copy; 2024 SEO Basics</p>
  </footer>
</body>
```
Diese Struktur macht Inhalte für User und Suchmaschinen besser lesbar.
