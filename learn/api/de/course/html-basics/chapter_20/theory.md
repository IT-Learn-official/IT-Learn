# Backend vs Frontend

Du hast die Begriffe Backend und Frontend vielleicht schon gehört. Aber was bedeuten sie?

Backend:
- ist der Teil der Website, den User nicht sehen.
- kümmert sich um Datenspeicherung, Serverlogik und App-Funktionalität.
- Beispiele: Datenbanken, serverseitige Programmiersprachen (z. B. Python, Node.js), APIs.

Frontend:
- ist der Teil, mit dem User interagieren.
- kümmert sich um das visuelle Layout, Design und die User Experience.
- Beispiele: HTML, CSS, JavaScript.
