# Listen

Listen organisieren zusammengehörige Punkte. Verwende `<ul>` für ungeordnete Listen (mit Punkten) und `<ol>` für geordnete Listen (nummeriert). Jeder Eintrag kommt in ein `<li>`-Element.

---

Beispiel für eine ungeordnete Liste:
```html
<ul>
  <li>Milch</li>
  <li>Brot</li>
  <li>Eier</li>
</ul>
```
Das sieht so aus:
- Milch
- Brot
- Eier

---

Beispiel für eine geordnete Liste:
```html
<ol>
  <li>Heize den Ofen auf 350°F vor.</li>
  <li>Mische die Zutaten.</li>
  <li>Backe 30 Minuten.</li>
</ol>
```
Das wird angezeigt als:
1. Heize den Ofen auf 350°F vor.
2. Mische die Zutaten.
3. Backe 30 Minuten.
