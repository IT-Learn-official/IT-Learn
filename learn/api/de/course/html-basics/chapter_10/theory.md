# Tabellen

Tabellen zeigen Daten in Zeilen und Spalten mit `<table>`. Zeilen erstellst du mit `<tr>`, Header-Zellen mit `<th>` und Datenzellen mit `<td>`.

---

Beispiel für eine einfache Tabelle:
```html
<table>
  <tr>
    <th>Name</th>
    <th>Alter</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Charlie</td>
    <td>35</td>
  </tr>
  <tr>
    <td>Diana</td>
    <td>28</td>
  </tr>
</table>
```

Die Ausgabe sieht ungefähr so aus:

| Name      | Alter |
|-----------|-------|
| Alice     | 30    |
| Bob       | 25    |
| Charlie   | 35    |
| Diana     | 28    |
