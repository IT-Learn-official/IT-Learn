# GitHub-Remotes: Clone, Push, Pull

Dein Repo kann an zwei Orten leben:

- lokal (dein Computer)
- remote (GitHub)

Clone = ein Remote-Repo herunterladen:

```bash
git clone <repo-url>
```

Push = deine Commits hochladen:

```bash
git push
```

Pull = neue Commits herunterladen:

```bash
git pull
```

---

Nach dem Clonen nennt Git das Remote meistens `origin`.
Du kannst das prüfen mit:

```bash
git remote -v
```
