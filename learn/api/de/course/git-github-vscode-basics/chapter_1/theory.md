# Versionsverwaltung, Git und GitHub

Wenn du programmierst, änderst du ständig Dateien.
Versionsverwaltung hält eine Historie fest, damit du zurückgehen kannst, wenn etwas schiefgeht.

**Git** ist das Werkzeug auf deinem Computer.
Es verfolgt Änderungen in einem Ordner, der **Repository** (Repo) heißt.

**GitHub** ist eine Website, die dein Git-Repo online speichern kann.
So hast du ein Backup und kannst mit anderen zusammenarbeiten.

---

Die zentrale Idee in Git ist ein **Commit**.
Ein Commit ist ein gespeicherter Schnappschuss mit einer kurzen Nachricht.

Begriffe, die du kennen solltest:

```text
Dateien, die du änderst -> Arbeitsverzeichnis
Gespeicherte Snapshots  -> Commits
Online-Kopie            -> Remote (GitHub)
```
