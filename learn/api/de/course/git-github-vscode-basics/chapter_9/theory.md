# Nützliche und unterhaltsame VS Code-Erweiterungen

## Visuell & UX

- [Dracula Official](https://marketplace.visualstudio.com/items?itemName=dracula-theme.theme-dracula) – Beliebtes, augenfreundliches Dark Theme mit gutem Kontrast für lange Coding-Sessions.
- [vscode-icons](https://marketplace.visualstudio.com/items?itemName=vscode-icons-team.vscode-icons) – Auffällige Icons für Dateien und Ordner; große Projekte lassen sich dadurch leichter scannen.
- [Peacock](https://marketplace.visualstudio.com/items?itemName=johnpapa.vscode-peacock) – Tönt das ganze VS Code-Fenster pro Projekt, damit du sofort erkennst, in welchem Arbeitsbereich du bist.
- [Indent Rainbow](https://marketplace.visualstudio.com/items?itemName=oderwat.indent-rainbow) – Färbt Einrückungsebenen, damit du Verschachtelungsfehler schnell siehst.

## Produktivität & Codequalität

- [Prettier – Code formatter](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) – Opinionated Formatter, der beim Speichern automatisch formatiert, damit Teams einen konsistenten Stil haben.
- [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) – Lintet JavaScript/TypeScript und findet häufige Bugs und Style-Probleme während du tippst.
- [GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens) – Erweitert die eingebaute Git-Ansicht mit Blame, Historie und Autorenschaft pro Zeile.
- [Path Intellisense](https://marketplace.visualstudio.com/items?itemName=christian-kohler.path-intellisense) – Autovervollständigung von Dateipfaden in Imports und requires.
- [Todo Tree](https://marketplace.visualstudio.com/items?itemName=Gruntfuggly.todo-tree) – Scannt deinen Code nach TODO/FIXME und zeigt die Treffer in einer Baumansicht.
- [Better Comments](https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments) – Markiert Kommentare (Alerts, Fragen, TODOs) in unterschiedlichen Farben.

## AI & smarte Vorschläge

- [GitHub Copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot) – AI Pair Programmer, der ganze Zeilen oder Funktionen anhand des Kontexts vorschlägt.
- [IntelliCode](https://marketplace.visualstudio.com/items?itemName=VisualStudioExptTeam.vscodeintellicode) – Microsofts AI-unterstützte Completion, die IntelliSense anhand häufiger Muster verbessert.

## Web Dev & JS/TS

- [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) – Startet einen lokalen Dev Server mit Live Reload für statische oder einfache Web-Projekte.
- [JavaScript (ES6) Code Snippets](https://marketplace.visualstudio.com/items?itemName=xabikos.JavaScriptSnippets) – Praktische Snippets für gängige JS/TS-Muster; spart Boilerplate.
- [Quokka.js](https://marketplace.visualstudio.com/items?itemName=WallabyJs.quokka-vscode) – „Scratchpad“ für JS/TS, das Code live ausführt und Ergebnisse inline anzeigt.

## Zusammenarbeit, APIs und DevOps

- [Live Share](https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare) – Live gemeinsam editieren und debuggen direkt in VS Code.
- [REST Client](https://marketplace.visualstudio.com/items?itemName=humao.rest-client) – HTTP-Requests aus `.http`-Dateien senden; praktisch zum API-Testen ohne Postman.
- [Docker](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-docker) – Container verwalten, Images ansehen und containerisierte Apps direkt im Editor debuggen.

## Regex, Markdown und Schreiben

- [Regex Previewer](https://marketplace.visualstudio.com/items?itemName=chrmarti.regex) – Side-by-side Ansicht, die Live-Matches zeigt, während du einen Regex bearbeitest.
- [Markdown All in One](https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one) – Shortcuts, Inhaltsverzeichnis und Auto-Formatierung für Markdown, alles in einer Extension.
- [:emojisense:](https://marketplace.visualstudio.com/items?itemName=bierner.emojisense) – Emoji-Autocomplete für Markdown, ausgelöst durch `:` plus Keyword.

## „Just for fun“ (aber trotzdem nützlich)

- [Error Lens](https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens) – Macht Errors und Warnings laut und inline sichtbar; Debugging fühlt sich fast wie ein Mini-Game an.
- [Polacode](https://marketplace.visualstudio.com/items?itemName=pnp.polacode) – Schöne Code-Screenshots für Doku oder Social Media.
- [Power Mode](https://marketplace.visualstudio.com/items?itemName=hoovercj.vscode-power-mode) – Particle-Effekte und Screen Shake beim Tippen, für Arcade-Vibes.
