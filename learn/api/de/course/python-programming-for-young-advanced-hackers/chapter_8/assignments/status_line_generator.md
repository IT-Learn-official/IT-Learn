# Mission: Status-Line-Generator

Ziel
- Erzeuge eine Statuszeile, die mehrere Variablen mit einem f-String kombiniert.

Schritte
1. Öffne `attachments/status_line_generator.py`.
2. Starte mit Variablen wie `filename`, `size_kb` und `status`.
3. Baue eine Zeile mit einem f-String und gib sie aus.

Hinweis
- Setze ein `f` vor den String: `f\"...{variable}...\"`.
