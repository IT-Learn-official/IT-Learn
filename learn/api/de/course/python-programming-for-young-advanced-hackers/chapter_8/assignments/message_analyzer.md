# Mission: Message-Analyzer

Ziel
- Analysiere eine geheime Nachricht mit Indexing, Slicing und Groß-/Kleinschreibung.

Schritte
1. Öffne `attachments/message_analyzer.py`.
2. Starte mit einem `message`-String im Code (z. B. `"Top Secret: HACK THE PLANET"`).
3. Gib aus:
   - das **erste** Zeichen (`message[0]`)
   - das **letzte** Zeichen (`message[-1]`)
   - einen Slice, der das geheime Wort enthält (`message[start:end]`)
4. Gib die komplette Nachricht einmal in `lowercase` und dann in `UPPERCASE` aus: `.lower()` und `.upper()`.
