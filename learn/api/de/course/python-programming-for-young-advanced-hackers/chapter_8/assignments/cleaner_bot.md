# Mission: Cleaner-Bot

Ziel
- Baue einen Bot, der „messy“ Text mit String-Methoden bereinigt.

Schritte
1. Öffne `attachments/cleaner_bot.py`.
2. Starte mit vorbereiteten Strings mit Extra-Spaces / unerwünschten Wörtern.
3. Verwende:
   - `.strip()` um Leerzeichen am Anfang/Ende zu entfernen
   - `.replace(old, new)` um Wörter zu verstecken oder zu ändern
   - `.split(\",\")` bei comma-separated Text und gib jedes Item in einer eigenen Zeile aus
