# Mission: Secret Name Tag

Ziel

- Erstelle einen geheimen Hacker-Namen aus Vor- und Nachnamen mit Slicing und einem f-String.

Schritte

1. Öffne `attachments/secret_name_tag.py`.
2. Lege `first_name` und `last_name` an.
3. Nutze Slicing, um einen Teil aus jedem Namen zu nehmen (z. B. die ersten 3 Buchstaben).
4. Kombiniere die Teile zu `code_name`.
5. Gib das Ergebnis mit einem f-String aus, z. B. `Your hacker name is: ...`.
