# Planung vs. Code markieren

Aufgabe

- Füge Inline‑Marker in `attachments/code_vs_programming_comments.py` hinzu.

Regeln

- Setze `# PROGRAMMING` ans Ende jeder Planungs-/Kommentarzeile.
- Setze `# CODE` ans Ende jeder ausführbaren Python‑Zeile.
- Keine Zeilen löschen oder umsortieren.
- Die Datei muss ohne Syntaxfehler ausführbar bleiben.

Validator‑Hinweis

- Der Validator erwartet, dass `name = getName(` und `print("Hello, " + name + "!")` mit den korrekten Markern vorhanden sind.
