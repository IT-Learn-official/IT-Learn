# Mission: Teile die Beute

Ziel
- Berechne, wie viele Münzen jede Person bekommt (mit `//`) und wie viele übrig bleiben (mit `%`).

Schritte
1. Öffne `attachments/share_the_loot.py`.
2. Setze `total_coins` und `players`.
3. Berechne:
   - `coins_per_player = total_coins // players`
   - `leftover_coins = total_coins % players`
4. Gib eine kurze, gut lesbare Zusammenfassung aus.

Checkliste
- [ ] `//` für `coins_per_player` verwendet.
- [ ] `%` für `leftover_coins` verwendet.
- [ ] Ergebnisse werden in verständlichen Sätzen ausgegeben.
- [ ] Script läuft ohne Fehler.
