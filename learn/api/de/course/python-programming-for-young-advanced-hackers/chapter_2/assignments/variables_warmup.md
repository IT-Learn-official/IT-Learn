# Mission: Gib deinen geheimen Zahlen Namen

Ziel

- Erstelle mindestens 3 numerische Variablen (mindestens 1 `int` und 1 `float`) und gib sie mit einem Label aus.

Schritte

1. Öffne `attachments/variables_warmup.py`.
2. Erstelle mindestens drei Variablen, zum Beispiel `lives = 3`, `speed = 1.5`, `coins = 10`.
3. Gib jede Variable sauber mit einem Label aus (mit Kommas in `print(...)` oder mit `str(...)`).

Checkliste

- [ ] Mindestens 3 Variablen erstellt.
- [ ] Mindestens 1 float verwendet (mit Punkt, z. B. `1.5`).
- [ ] Jede Variable wird mit einem Label ausgegeben.
- [ ] Script läuft ohne Fehler.
