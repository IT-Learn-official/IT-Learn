# Mission: Klammern-Power-Up

Ziel
- Zeige, dass Klammern `()` die Reihenfolge der Operationen verändern.

Schritte
1. Öffne `attachments/precedence_playground.py`.
2. Füge Print-Zeilen mit Paaren von Ausdrücken hinzu: einmal ohne Klammern und einmal mit Klammern.
3. Füge mindestens 2 zusätzliche Paare hinzu, davon mindestens 1 mit `//` (Ganzzahldivision) oder `%` (Rest).

Checkliste
- [ ] Mindestens 2 Ausdrucks-Paare hinzugefügt.
- [ ] Mindestens 2 zusätzliche Paare hinzugefügt.
- [ ] Mindestens 1 Beispiel verwendet `//` oder `%`.
- [ ] Script läuft ohne Fehler.
