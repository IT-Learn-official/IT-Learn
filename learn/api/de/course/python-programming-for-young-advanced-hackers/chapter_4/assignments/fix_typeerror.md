# Mission: Fix den TypeError

Ziel
- Repariere den Starter-Code in `attachments/fix_typeerror.py`, der Strings und Zahlen mischt und deshalb mit `TypeError` abstürzt.

Schritte
1. Öffne `attachments/fix_typeerror.py`.
2. Wandle zahlenähnliche Strings vor dem Rechnen in echte Zahlen um (mit `int()` oder `float()` wo nötig).
3. Wenn du Text und Zahlen gemeinsam ausgeben willst:
   - verwende `str(...)`, oder
   - verwende Kommas in `print(...)`.
4. Ändere die Story oder die Zahlen nicht; repariere nur die Typen.

Checkliste
- [ ] Script läuft ohne Fehler.
- [ ] Output ist plausibel (z. B. 16, 150, 15 wie in der Aufgabe gedacht).
- [ ] `int()`/`float()` fürs Rechnen und `str()`/Kommas für Text + Zahl verwendet.
