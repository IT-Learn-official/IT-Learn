# Mission: Boolean-Stimmungslicht

Ziel

- Zeige mit `bool(...)`, welche Werte als True/False gelten.

Schritte

1. Öffne `attachments/boolean_mood_light.py`.
2. Erstelle oder verwende mindestens 4 Variablen mit unterschiedlichen Werten (z. B. `True`, `""`, `0`, `"hi"`).
3. Drucke pro Variable:
   - den Namen
   - den Wert
   - `bool(wert)`
4. Füge mindestens 1 eigene Variable hinzu und drucke sie ebenfalls.

Tipp

- Verwende `repr(...)` bei Strings, wenn du leer vs. nicht-leer klar sehen willst.

Checkliste

- [ ] Mindestens 4 Zeilen Output.
- [ ] Jede Zeile zeigt einen Wert und `bool(...)`.
- [ ] Script läuft ohne Fehler.
