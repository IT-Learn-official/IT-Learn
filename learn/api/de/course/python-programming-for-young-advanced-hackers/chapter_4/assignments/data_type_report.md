# Mission: Datentyp-Report

Ziel

- Erstelle einen kleinen Report, der pro Variable Wert und Typ mit `type()` zeigt.

Schritte

1. Öffne `attachments/data_type_report.py`.
2. Erstelle mindestens 4 Variablen mit unterschiedlichen Typen:
   - 1 `str`
   - 1 `int`
   - 1 `float`
   - 1 `bool`
3. Drucke pro Variable eine Zeile mit:
   - Name
   - Wert
   - Typ (mit `type(...)`)

Checkliste

- [ ] Mindestens 4 Variablen, alle mit unterschiedlichen Typen.
- [ ] `type(...)` für jede Variable verwendet.
- [ ] Mindestens 4 klare Zeilen Output.
- [ ] Script läuft ohne Fehler.
