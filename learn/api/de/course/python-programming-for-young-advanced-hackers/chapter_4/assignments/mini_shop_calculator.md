# Mission: Mini-Shop-Rechner

Ziel
- Baue einen Mini-Shop-Rechner, der Preis und Menge (als Strings) in Zahlen umwandelt und einen Zwischensumme berechnet.

Schritte
1. Öffne `attachments/mini_shop_calculator.py`.
2. Wandle `price_str` mit `float()` in einen float um.
3. Wandle `quantity_str` mit `int()` in einen int um.
4. Berechne `subtotal = price * quantity`.
5. Gib die Zwischensumme klar aus (z. B. `Subtotal: 29.97`).

Optional
- Füge `discount_active = True/False` hinzu und gib eine zweite Zeile aus, wenn der Rabatt aktiv ist.

Checkliste
- [ ] `float()` für den Preis verwendet.
- [ ] `int()` für die Menge verwendet.
- [ ] Zwischensumme korrekt berechnet und ausgegeben.
- [ ] Script läuft ohne Fehler.
