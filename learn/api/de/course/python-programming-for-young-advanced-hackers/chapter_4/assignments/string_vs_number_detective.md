# Mission: String vs. Number Detective

Ziel

- Wandle Werte, die wie Zahlen aussehen (aber Strings sind), in echte Zahlen um, mache mindestens 1 Rechnung und gib alles ohne `TypeError` aus.

Schritte

1. Öffne `attachments/string_vs_number_detective.py`.
2. Wandle mindestens 2 String-Werte mit `int()` (oder `float()`) in Zahlen um.
3. Mache mindestens 1 Rechnung, z. B. `total_score = score + bonus`.
4. Drucke klar beschriftete Zeilen, z. B. `Age: 13`, `Score: 250`, `Total score: 255`.
5. Drucke außerdem mindestens 1 freundlichen Satz, der Text und Zahlen kombiniert (mit `str()` oder Kommas).

Checkliste

- [ ] Mindestens 2 Konvertierungen String -> Zahl.
- [ ] Mindestens 1 numerische Berechnung.
- [ ] Text + Zahl gemischt ohne Fehler (`str()` oder Kommas).
- [ ] Script läuft ohne Fehler.
