# Mission: Trace den Score

Ziel
- Trace, wie sich `score` verändert, indem du nach jedem Update `print(...)` hinzufügst.

Schritte
1. Öffne `attachments/trace_the_score.py`.
2. Füge nach jeder Änderung des Scores ein `print(...)` hinzu.
3. Führe den Code aus und stelle sicher, dass die Ausgabe exakt so ist:

```
Score: 0
Score: 10
Score: 15
```
