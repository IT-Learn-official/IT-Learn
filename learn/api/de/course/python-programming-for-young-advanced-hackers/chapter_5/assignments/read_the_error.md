# Mission: Lies den Fehler

Ziel
- Lies die Fehlermeldung, finde den Tippfehler und repariere den Variablennamen, sodass das Script ohne Fehler läuft.

Schritte
1. Öffne `attachments/read_the_error.py`.
2. Führe den Code aus und lies die Fehlermeldung.
3. Behebe den Tippfehler im Variablennamen, damit der Wert korrekt ausgegeben wird.
