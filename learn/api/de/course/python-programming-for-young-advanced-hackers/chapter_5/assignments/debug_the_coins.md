# Mission: Debug die Coins

Ziel
- Nutze Print-Debugging, um den Wert von `coins` zu tracen und die Updates zu reparieren.

Schritte
1. Öffne `attachments/debug_the_coins.py`.
2. Füge nach jeder Änderung von `coins` ein `print(...)` hinzu.
3. Repariere die Fehler so, dass die Ausgabe exakt so ist:

```
Start coins: 5
After buying a shield: 3
After finding treasure: 8
```
