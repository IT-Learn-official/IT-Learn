# Mission: Super-Rechner - Potenzen und Wurzeln

Ziel
- Baue einen Mini-Rechner, der Potenzen und Wurzel für eine positive Zahl berechnet.

Schritte
1. Öffne `attachments/super_calculator_powers_roots.py`.
2. Frage eine positive Zahl ab.
3. Berechne und gib aus:
   - Quadrat: `n ** 2`
   - Kubik: `n ** 3`
   - Quadratwurzel: `math.sqrt(n)`

Hinweise
- `import math`
