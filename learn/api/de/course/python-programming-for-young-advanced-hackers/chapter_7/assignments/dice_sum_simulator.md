# Mission: Würfel - Summe vieler Würfe

Ziel

- Simuliere einen 6-seitigen Würfel und addiere alle Würfe zusammen.

Schritte

1. Öffne `attachments/dice_sum_simulator.py`.
2. Frage den Benutzer, wie oft gewürfelt werden soll.
3. Verwende `random.randint(1, 6)` für jeden Wurf.
4. (Optional) Gib jeden Wurf aus, z. B. `Roll 1: 4`.
5. Halte ein `total` und gib am Ende die Gesamtsumme aus.

Hinweise

- `import random`
- Starte mit `total = 0` vor der Schleife.
