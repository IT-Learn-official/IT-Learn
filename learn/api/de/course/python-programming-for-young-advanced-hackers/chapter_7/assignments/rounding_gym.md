# Mission: Rounding-Gym - round, floor und ceil

Ziel

- Trainiere das Runden mit `round`, `math.floor` und `math.ceil`.

Schritte

1. Öffne `attachments/rounding_gym.py`.
2. Frage eine Dezimalzahl ab (z. B. `3.7` oder `-1.2`).
3. Gib klar aus:
   - Rounded: `round(x)`
   - Floor: `math.floor(x)`
   - Ceil: `math.ceil(x)`

Hinweise

- `import math`
- floor geht immer nach unten, ceil immer nach oben.
