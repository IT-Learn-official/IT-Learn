# Mission: Zuweisungsoperatoren-Lab

Ziel

- Übe Zuweisungsoperatoren: `=`, `+=`, `-=`, `*=`, `/=`.

Schritte

1. Öffne `attachments/assignment_operators.py`.
2. Starte mit einer Variable und aktualisiere sie mit jedem Operator.
3. Gib den Wert nach jeder Änderung aus.
4. Sorge für mindestens 5 `print(...)`-Zeilen (eine pro Operator).
