# Mission: Vergleichsoperatoren-Detektiv

Ziel
- Teste alle Vergleichsoperatoren: `==`, `!=`, `<`, `>`, `<=`, `>=`.

Schritte
1. Öffne `attachments/comparison_operators.py`.
2. Verwende jeden Operator, um zwei Zahlen zu vergleichen, und gib das Ergebnis aus.
3. Sorge für mindestens 6 `print(...)`-Zeilen (eine pro Operator).
