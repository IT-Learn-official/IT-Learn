# Mission: Rechenoperatoren-Playground

Ziel
- Probiere alle Rechenoperatoren aus: `+`, `-`, `*`, `/`, `//`, `%` und `**`.

Schritte
1. Öffne `attachments/arithmetic_operators.py`.
2. Verwende jeden Operator mit mindestens zwei Zahlen und gib das Ergebnis aus.
3. Sorge für mindestens 6 `print(...)`-Zeilen (eine pro Operator).
