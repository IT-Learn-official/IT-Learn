# Mission: Logische-Operatoren-Challenge

Ziel

- Probiere die logischen Operatoren: `and`, `or`, `not`.

Schritte

1. Öffne `attachments/logical_operators.py`.

3. Sorge für mindestens 3 `print(...)`-Zeilen (eine pro Operator).
