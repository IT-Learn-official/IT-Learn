# Übung: Funktions-Combo Mini-Report

Ziel

- Kombiniere mehrere Built-in-Funktionen in einem kleinen "Status Report" mit `print(...)`.

Schritte

1. Öffne `attachments/function_combo_report.py`.
2. Verwende diese Funktionen jeweils mindestens 1-mal:
   - `len`
   - `abs`
   - `round`
   - `type`
3. Drucke einen Report mit mehreren Zeilen (Name, Länge, Abstand, gerundeter Wert, Typ, ...).
4. Du kannst gern zusätzliche Variablen und Zeilen hinzufügen.

Checkliste

- [ ] `len`, `abs`, `round` und `type` alle verwendet.
- [ ] Klaren mehrzeiligen Report gedruckt.
- [ ] Script läuft ohne Fehler.
