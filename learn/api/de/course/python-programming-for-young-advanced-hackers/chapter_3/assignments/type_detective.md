# Übung: Type-Detektiv mit type

Ziel

- Nutze `type(...)`, um zu sehen, welchen Typ eine Variable hat (z. B. `int`, `float`, `str`).

Schritte

1. Öffne `attachments/type_detective.py`.
2. Verwende `type(...)` bei mindestens 3 verschiedenen Variablen.
3. Drucke pro Variable einen klaren Satz, z. B.: `coins ist vom Typ <...>`.
4. (Extra) Rate den Typ vor dem Ausführen und prüfe danach, ob du richtig lagst.

Checkliste

- [ ] `type(...)` bei mindestens 3 Variablen verwendet.
- [ ] Typ-Information mit klaren Messages ausgegeben.
- [ ] Du kannst den Unterschied zwischen `int`, `float` und `str` erklären.
