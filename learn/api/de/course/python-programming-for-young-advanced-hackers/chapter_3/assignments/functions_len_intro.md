# Übung: Zeichen zählen mit len

Ziel

- Nutze `len(...)`, um die Länge von Strings zu messen, speichere die Ergebnisse in Variablen und gib sie sauber aus.

Schritte

1. Öffne `attachments/functions_len_intro.py`.
2. Rufe `len(...)` mindestens 3-mal mit unterschiedlichen Strings auf (z. B. `"cat"`, `"dragon"`, `"Hello, world!"`).
3. Speichere jedes Ergebnis in einer Variable (z. B. `length_cat = len("cat")`).
4. Gib pro String einen verständlichen Satz aus, der zeigt, was du gemessen hast.

Checkliste

- [ ] `len(...)` mindestens 3-mal verwendet.
- [ ] Jedes Ergebnis in einer Variable gespeichert.
- [ ] Klare `print(...)`-Ausgaben hinzugefügt.
- [ ] Script läuft ohne Fehler und zeigt die richtigen Längen.
