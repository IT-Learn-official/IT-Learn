# Übung: Scoreboard mit round runden

Ziel

- Nutze `round(...)`, um "unsaubere" Dezimalzahlen lesbarer zu machen.

Schritte

1. Öffne `attachments/round_scoreboard.py`.
2. Runde mindestens eine Zahl auf eine ganze Zahl (ein Argument: `round(x)`).
3. Runde mindestens eine weitere Zahl auf 2 Nachkommastellen (zwei Argumente: `round(x, 2)`).
4. Speichere die gerundeten Werte in Variablen.
5. Drucke ein kleines Scoreboard mit Original- und Rundungswerten.

Checkliste

- [ ] `round(x)` verwendet (1 Argument).
- [ ] `round(x, n)` verwendet (2 Argumente).
- [ ] Ergebnisse in Variablen gespeichert.
- [ ] Klare Scoreboard-Ausgabe gedruckt.
