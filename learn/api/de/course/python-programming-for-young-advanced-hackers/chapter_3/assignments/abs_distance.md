# Übung: Abstand zu 0 mit abs

Ziel

- Nutze `abs(...)`, um den Abstand von Zahlen zu 0 zu berechnen (immer 0 oder positiv).

Schritte

1. Öffne `attachments/abs_distance.py`.
2. Verwende `abs(...)` mindestens 2-mal: einmal für eine negative und einmal für eine positive Zahl.
3. Speichere jedes Ergebnis in einer Variable (z. B. `distance1 = abs(step1)`).
4. Gib pro Zahl einen klaren Satz aus: Originalwert und Abstand.

Checkliste

- [ ] `abs(...)` mindestens 2-mal verwendet.
- [ ] Jedes Ergebnis in einer Variable gespeichert.
- [ ] Klare Ausgabe mit Originalwert + Abstand gedruckt.
- [ ] Script läuft ohne Fehler.
