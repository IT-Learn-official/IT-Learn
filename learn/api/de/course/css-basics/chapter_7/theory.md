# Links gestalten

Links können je nach Aktion des Users unterschiedlich aussehen.

`a` ist der Normalzustand.
`a:hover` gilt, wenn die Maus über dem Link ist.
`a:visited` ist ein Link, den der User schon geöffnet hat.

```css
a {
  color: #0b63f6;
  text-decoration: none;
}

a:hover {
  color: #0847b8;
  text-decoration: underline;
}
```

So wirken Links interaktiver.
