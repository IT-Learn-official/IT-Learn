# Textausrichtung und Abstände

Manchmal wirkt Text „komisch“, obwohl die Rechtschreibung passt.
Meistens sind die Abstände das Problem.

`line-height` fügt Abstand zwischen Zeilen hinzu.
`text-align` richtet Text links, zentriert oder rechts aus.
`letter-spacing` passt den Abstand zwischen Buchstaben an.

```css
p {
  line-height: 1.6;
  letter-spacing: 0.2px;
}

h2 {
  text-align: center;
}
```

Schon kleine Änderungen bei den Abständen machen das Lesen deutlich leichter.
