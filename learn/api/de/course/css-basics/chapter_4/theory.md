# Farben und Einheiten

CSS gibt dir verschiedene Möglichkeiten, Farben zu schreiben.
Du kannst einen Namen wie `red` verwenden, einen HEX-Code wie `#ff0000`, RGB oder HSL.
Das funktioniert alles.

Größen haben auch unterschiedliche Einheiten.
`px` ist fix.
`%` ist relativ.
`em` und `rem` orientieren sich an der Schriftgröße.

```css
.box {
  background-color: #4caf50;
  width: 50%;
  font-size: 1rem;
}
```

Du musst nicht jede Einheit am ersten Tag beherrschen.
Starte mit `px`, `%` und `rem`.
