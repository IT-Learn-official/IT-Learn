# CSS organisieren

Wenn CSS größer wird, zählt Organisation.

Versuche zusammengehörige Regeln zu gruppieren, nutze Kommentare und halte die Formatierung sauber.

```css
/* Buttons */
.btn-primary {
  background: #2d7ff9;
  color: white;
}
```

Sauberes CSS lässt sich später viel leichter aktualisieren.
