# Hintergründe hinzufügen

Hintergründe helfen dabei, Bereiche voneinander abzuheben.

Du kannst eine einfache Farbe, einen Verlauf oder ein Bild verwenden.

```css
.hero {
  background: linear-gradient(90deg, #4facfe, #00f2fe);
}

.banner {
  background-image: url("stars.jpg");
  background-size: cover;
  background-position: center;
}
```

`cover` sorgt dafür, dass das Bild den Bereich vollständig ausfüllt.
