# Was ist Responsive Design?

Responsive Design bedeutet, dass deine Seite am Handy, Tablet und Desktop gut aussieht.

Das Layout soll sich anpassen, wenn sich die Bildschirmgröße ändert.

```css
img {
  max-width: 100%;
  height: auto;
}
```

So bleiben Bilder auf kleinen Screens innerhalb ihres Containers.
