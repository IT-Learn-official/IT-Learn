# Selektoren und Properties verstehen

Eine CSS-Regel sieht so aus:

```css
selector {
  property: value;
}
```

Einfach erklärt:

- selector = was du auswählst
- property = was du änderst
- value = die neue Einstellung
- declaration = eine Zeile (Beispiel: `color: blue;`)

Beispiel:

```css
h1 {
  color: navy;
  font-size: 30px;
}
```

Hier:

- h1 wählt alle h1-Überschriften aus
- color ändert die Textfarbe
- font-size ändert die Textgröße
