# Was ist CSS und warum verwendet man es?

Wenn du eine reine HTML-Seite öffnest, funktioniert sie, aber sie wirkt meistens langweilig.
Hier kommt CSS ins Spiel.

CSS ist der Teil, der deiner Seite Stil gibt.
Es verändert Farben, Schriftgrößen, Abstände und Layout.

Wenn HTML die Struktur ist, ist CSS das Design.

Wenn dir HTML noch neu vorkommt, starte am besten hier:
https://itlearn.be/learn/#/courses?courseId=html-basics

---

Kleines Beispiel:

```html
<h1 class="title">Hello</h1>
```

```css
.title {
  color: blue;
}
```

Diese eine CSS-Regel macht den Titel blau.
