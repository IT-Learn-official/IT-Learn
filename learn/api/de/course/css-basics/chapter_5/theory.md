# Grundlegendes Text-Styling

Text ist das Erste, was Menschen lesen, daher ist gutes Styling wichtig.

Die häufigsten Text-Einstellungen sind:

- `font-family`
- `font-size`
- `font-weight`
- `color`

```css
body {
  font-family: Arial, sans-serif;
  font-size: 16px;
  color: #222;
}

h1 {
  font-weight: 700;
}
```

Die letzte Zeile macht die Überschrift fett.
