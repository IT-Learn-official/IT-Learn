# Das CSS-Box-Model erklärt

In CSS wird jedes Element wie eine Box behandelt.

Diese Box hat vier Teile:
content, padding, border und margin.

```css
.card {
  padding: 16px;
  border: 2px solid #333;
  margin: 20px;
}
```

Stell dir ein Gemälde im Rahmen vor. Du kannst es in 4 Teile teilen:

- Das Gemälde selbst (der content)
- Der Abstand zwischen Gemälde und Rahmen (das padding)
- Der Rahmen (der border)
- Die Wand außen herum (der margin)
