# Head & Meta

The `<head>` holds metadata, information about the page that browsers and search engines use. It doesn't show up as page content, but it does a lot.

---

Main items in the head:
- `<title>`: this is the text shown on a tab in your browser. This is most of the time the title of the page.
- `<meta charset="utf-8">`: tells the browser which characters you use so text displays correctly.
- `<meta name="viewport" content="width=device-width, initial-scale=1">`: makes pages responsive on phones.

---

Think of the head as the label on the package you ordered on Amazon. It doesn't change the inside, but it helps the delivery company know how to handle it, where to send it, and what precautions to take.
