# Headings

Headings are the big titles on a page. They range from `<h1>` (big title) to `<h6>` (smaller subtitle).
Here are some examples of how headings look like:

# H1

## H2

### H3

#### H4

##### H5

###### H6

---

Fun fact: Search engines and screen readers use headings to understand the structure of your page.

Important rules:

- Prefer a single page-level `<h1>` on most pages (usually the main page title).
- Choose heading levels based on document structure and semantic nesting (`<h1>`, `<h2>`, `<h3>`, etc.), not visual size.
- Keep structure and styling separate: use semantic headings for meaning, and use CSS to control appearance.
- It is okay to deviate from one `<h1>` only when there is a valid structural reason.
