# Backend VS Frontend

You may have heard the terms backend and frontend before. But what do they mean?

Backend:
- Is the part of the website users don't see.
- Handles data storage, server logic, and application functionality.
- Examples: databases, server-side programming languages (like Python, Node.js), APIs.

Frontend:
- Is the part of the website users interact with.
- Handles the visual layout, design, and user experience.
- Examples: HTML, CSS, JavaScript.
