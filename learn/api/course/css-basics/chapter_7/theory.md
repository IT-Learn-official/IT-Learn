# Styling Links

Links can have different looks depending on what the user does.

`a` is the normal state.
`a:hover` is when the mouse is over the link.
`a:visited` is a link the user already opened.

```css
a {
  color: #0b63f6;
  text-decoration: none;
}

a:hover {
  color: #0847b8;
  text-decoration: underline;
}
```

This makes links feel more interactive.
