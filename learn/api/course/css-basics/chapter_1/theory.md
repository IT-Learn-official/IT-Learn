# What Is CSS and Why Use It?

When you open a plain HTML page, it works, but it usually looks boring.
That is where CSS comes in.

CSS is the part that gives your page style.
It changes colors, text sizes, spacing, and layout.

So if HTML is the structure, CSS is the design.

If HTML still feels new, start here first:
https://itlearn.be/learn/#/courses?courseId=html-basics

---

Small example:

```html
<h1 class="title">Hello</h1>
```

```css
.title {
  color: blue;
}
```

That one CSS rule turns the title blue.
