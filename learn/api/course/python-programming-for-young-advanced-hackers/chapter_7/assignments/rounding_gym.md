# Mission: Rounding Gym  Round, Floor, and Ceil

Train your rounding skills in the "rounding gym".

## What your program should do

1. Ask the user to enter a **decimal number** (like `3.7` or `-1.2`).
2. Compute and print:
   - The number **rounded** to the nearest whole number using `round`.
   - The **floor** using `math.floor`.
   - The **ceil** using `math.ceil`.
3. Label each result clearly.

## Hints

- Use `import math` at the top of your file.
- Remember:
  - `round(x)` uses normal rounding rules.
  - `math.floor(x)` always goes **down**.
  - `math.ceil(x)` always goes **up**.

## Example (one possible run)

Input:

```text
Enter a decimal number: 3.7
```

Possible output:

```text
Rounded: 4
Floor: 3
Ceil: 4
```
