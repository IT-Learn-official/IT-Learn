# Mission: Custom Greeting with Default

Write a function with a default parameter value.

## What your program should do

1. Define a function `greet(name, greeting="Hello")`.
2. When called with one argument (just `name`), it should use the default greeting.
3. When called with two arguments (`name` and `greeting`), it should use the custom greeting.
4. Call the function at least twice:
   - Once like `greet("Alex")`.
   - Once like `greet("Mia", "Hi")`.

## Hints

- The default value goes in the function definition.
- Inside the function, print both `greeting` and `name`.
