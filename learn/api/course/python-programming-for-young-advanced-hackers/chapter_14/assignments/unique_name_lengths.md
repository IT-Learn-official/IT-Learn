# Mission: Unique Name Lengths (Set Comprehension)

Use a set comprehension to find all different name lengths in a list.

## What your program should do

1. Start with a list of names, for example: `["Mia", "Alex", "Zed", "Mia"]`.
2. Use a **set comprehension** to build `lengths`, a set of the different lengths of the names.
3. Print the set.

## Hints

- Set comprehension pattern: `{expression for item in iterable}`.
- Use `len(name)` inside the comprehension.
