# Mission: Create a Book Class

Turn book data into a simple class.

## What your program should do

1. Define a class `Book`.
2. In `__init__(self, title, pages)`, store the values in `self.title` and `self.pages`.
3. Add a method `description(self)` that returns a string like `"Python for Hackers (120 pages)"`.
4. Create at least one `Book` object and print the result of `description()`.

## Hints

- Remember to use `self.title = title` and `self.pages = pages` in `__init__`.
- `description` should use `return`, not `print`, so you can print its result outside the class.
